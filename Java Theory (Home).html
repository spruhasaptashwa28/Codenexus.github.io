PROGRAMMING WORLD OF 'JAVA'





1. Introduction to Java :


⋄ Java is a popular programming language created by James Gosling in 1995.

⋄ It is known for being platform-independent, meaning it can run on different devices using the Java Virtual Machine (JVM).

⋄ Java follows the Write Once, Run Anywhere (WORA) principle.



2. Key Features of Java :


⋄ Object-Oriented: Java is based on classes and objects.

⋄ Platform-Independent: Code written in Java can run on any device with JVM.

⋄ Simple and Easy to Learn: Java’s syntax is clear and similar to C++.

⋄ Robust: Java handles errors efficiently using features like exception handling.

⋄ Multithreaded: Java can run multiple tasks at the same time.

⋄ High Performance: Java’s Just-In-Time (JIT) compiler improves speed.

⋄ Distributed Computing: Java can easily develop network-based applications.



3. Java Architecture :


Java architecture involves :


⋄ Java Source Code (.java) → Written by the programmer

⋄ Compiler (javac) → Converts code to bytecode (.class)

⋄ JVM (Java Virtual Machine) → Executes the bytecode

⋄ JVM acts as an interpreter, making Java platform-independent.



4. Java Program Structure :


A typical Java program includes:


⋄ Class Declaration → Defines the structure

⋄ Main Method → Entry point for program execution

⋄ Statements → Instructions for the computer


Example :

                //Simple Java Program
                    class Greeting {
                        public static void main(String[] args) {
                            System.out.println("Welcome to Java Programming!");
                        }
                    }


5. Core Concepts in Java :


⋄ Class: A blueprint for objects that defines properties and methods.

⋄ Object: An instance of a class that holds data and behaviors.

⋄ Inheritance: Enables one class to inherit properties from another.

⋄ Polymorphism: Allows one method to behave differently based on input.

⋄ Encapsulation: Protects data by wrapping it inside classes and controlling access.

⋄ Abstraction: Hides complex logic and shows only essential details.



6. Data Types in Java :


⋄ Primitive Types: int, float, char, boolean, etc.

⋄ Non-Primitive Types: String, Arrays, Classes, etc.


Example of Variables:


                int age = 25; 
                double pi = 3.14; 
                char grade = 'A'; 
                String name = "Java";


7. Control Statements :


Java includes decision-making and looping structures such as:


⋄ if-else (for conditions)

⋄ switch (for multiple options)

⋄ for, while, and do-while (for loops)
