C Programming Basics

Introduction to C Programming

The C programming language is a powerful and widely-used language developed by Dennis Ritchie in 1972. Known for its efficiency and flexibility, C is used for system programming, operating systems, and embedded applications.

Data Types in C

Data Type Description Example
int Stores integers (whole numbers) int a = 10;
float Stores floating-point numbers float pi = 3.14;
char Stores single characters char grade = 'A';
double Stores double-precision floating numbers double price = 99.99;

Variables in C

Variables are used to store data values. They can be declared and initialized as shown below:

#include <stdio.h> int main() { int age = 21; float height = 5.9; char grade = 'A'; printf("Age: %d\n", age); printf("Height: %.1f feet\n", height); printf("Grade: %c\n", grade); return 0; }

Input and Output in C

Use printf() for displaying output and scanf() for accepting user input.

#include <stdio.h> int main() { int num; printf("Enter a number: "); scanf("%d", &num); printf("You entered: %d\n", num); return 0; }
