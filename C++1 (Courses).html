Variables in C++
Theory: Variables are containers for storing data values. In C++, each variable must be declared with a data type before use. 
#include 
using namespace std;

int main() {
    int age = 25;
    float pi = 3.14;
    char grade = 'A';

    cout << "Age: " << age << endl;
    cout << "Pi value: " << pi << endl;
    cout << "Grade: " << grade << endl;

    return 0;
}
Output: Age: 25 Pi value: 3.14 Grade: A

Control Structures in C++
Theory: Control structures control the flow of a program. Common examples include *if-else, **switch, **for, **while, and **do-while* loops. 
#include 
using namespace std;

int main() {
    int num;
    cout << "Enter a number: ";
    cin >> num;

    if (num % 2 == 0) {
        cout << num << " is even." << endl;
    } else {
        cout << num << " is odd." << endl;
    }

    return 0;
}
Output: Enter a number: *5* 5 is odd. (OR) Enter a number: *4* 4 is even.
