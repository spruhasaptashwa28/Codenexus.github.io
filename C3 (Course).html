Functions in C

Introduction to Functions

A function is a block of code designed to perform a specific task. Functions improve code reusability and modular programming.

Syntax of a Function:

return_type function_name(parameters) { // Code block return value; }

Built-in Functions

C language provides many built-in functions such as printf(), scanf(), and sqrt().

#include <stdio.h> #include <math.h> int main() { double result = sqrt(25); printf("Square root of 25 is: %.2f\n", result); return 0; }

User-defined Functions

User-defined functions are custom functions created by the programmer for specific tasks.

#include <stdio.h> // Function Declaration int add(int a, int b); int main() { int x = 5, y = 10; int sum = add(x, y); // Function Call printf("Sum = %d\n", sum); return 0; } // Function Definition int add(int a, int b) { return a + b; }

Recursive Functions

A recursive function calls itself until a termination condition is met.

#include <stdio.h> // Recursive Function for Factorial int factorial(int n) { if (n == 0) return 1; else return n * factorial(n - 1); } int main() { int num = 5; printf("Factorial of %d is %d\n", num, factorial(num)); return 0; }

Function Pointers

A function pointer stores the address of a function and allows dynamic execution.

#include <stdio.h> // Function Declaration void greet() { printf("Hello from Function Pointer!\n"); } int main() { void (*ptr)() = greet; // Function Pointer Declaration ptr(); // Function Call via Pointer return 0; }
